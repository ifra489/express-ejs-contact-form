# Express EJS Contact Form

A simple contact form project built using Node.js, Express.js, and EJS Template Engine.

This project demonstrates:
- Express server setup
- EJS template rendering
- Form handling using POST requests
- Receiving form data with req.body
- Dynamic rendering of submitted data

---

# 🚀 Features

- Dynamic EJS templates
- Contact form with validation
- Form submission handling
- Success page rendering
- Beginner-friendly project structure

---

# 🛠 Technologies Used

- Node.js
- Express.js
- EJS
- HTML5

# ⚙️ Installation


Clone the repository:
```
git clone https://github.com/ifra489/express-ejs-contact-form.git
```
Move into the project folder:
```
cd express-ejs-contact-form
```
Install dependencies:
```
npm install
```
Run the server:
```
node app.js
```
---

# 🌐 Run in Browser

http://localhost:3000/contact

# 📷 Project Workflow

1. User opens contact form
2. User enters details
3. Form data is sent to backend
4. Server processes data
5. Success page displays submitted information
